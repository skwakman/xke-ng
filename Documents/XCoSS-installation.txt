
  XCoSS installation description
===========================================

- Installeer Java6

- Installeer Tomcat6

- Installeer MySQL 5.5

- Installeer MySQL Workbench 5.2 CE (optioneel)

- Installeer MySQL Connector/J 5.1.x

- Configureer Tomcat:

  - Voeg toe aan ./conf/context.xml

	<Resource name="jdbc/xcossds"
			auth="Container"
			type="javax.sql.DataSource"
			username="xcoss"
			password="xcoss"
			driverClassName="com.mysql.jdbc.Driver"
			url="jdbc:mysql://localhost:3306/xcoss"
			maxActive="10"
			maxIdle="4"
	/>
  
  - Plaats mysql-connector-java-5.1.x-bin.jar in ./lib/

- Configureer MySQL:

  - Start: mysql -u root -p

  - Voer uit: 
    create database xcoss default character set utf8 default collate utf8_general_ci;
	grant all privilegest on xcoss.* to xcoss@'localhost';
	set password for xcoss@'localhost' = old_password('xcoss');
